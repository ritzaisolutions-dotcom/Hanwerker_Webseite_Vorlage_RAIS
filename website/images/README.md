# Bilder — Handwerker-Vorlage

Dateinamen sind **fix**. Kundenfotos unter diesen Namen ablegen (WebP + JPG/PNG empfohlen).

| Datei | Verwendung |
|-------|------------|
| `logo-client.png` / `.svg` | Header, Hero |
| `favicon-client.png` | Browser-Tab |
| `hero-video.mp4` | Hero-Hintergrund (optional) |
| `hero-poster.jpg` / `.svg` | Video-Fallback |
| `vorher_slide.jpg` / `.svg` | Vorher/Nachher-Slider |
| `nachher_slide.jpg` / `.svg` | Vorher/Nachher-Slider |
| `ref_1` … `ref_10` | Referenzen-Grid |
| `team-bild` | Team-Seite |
| `og-image.jpg` | Social Share (1200×630) |
| `partner/*.png` | Partner-Karussell (siehe `partner/README.md`) |

Platzhalter-SVGs sind für lokale Demo-Vorschau. Vor Go-Live durch echte Assets ersetzen.
