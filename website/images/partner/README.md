# Partner-Logos

1. In `website/js/config.js` → `partner[]` eintragen (`name`, `url`, `logo`)
2. Logos hier ablegen, z. B. `partner/brillux.png` (PNG/WebP, transparenter Hintergrund, ca. 200×80px)
3. Oder einmalig laden: `node scripts/fetch-partner-logos.js` (Domains im Skript anpassen)

Leerer `logo`-String → Text-Badge im Karussell.
