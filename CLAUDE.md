# CLAUDE.md — Handwerker Website Template

Dieses Repository ist ein **branchenagnostisches Template** für Handwerker-Websites.
Beim Onboarding eines neuen Kunden: `SETUP.md` lesen und folgen.

## Konfiguration
Alle kundespezifischen Werte: `website/js/config.js`
Automatische DOM-Befüllung: `website/js/config-apply.js`

## Tech Stack
Vanilla HTML5 / CSS3 / ES6+. Kein Framework. Keine npm-Abhängigkeiten zur Laufzeit.

## SOP
`sop-webseiten-bau/` — vollständiges Playbook für Demo- und Live-Build.
Bei Unklarheiten: 13-claude-rules.md und 14-learnings.md zuerst lesen.

## Deploy
Root Directory in Vercel: `website` — Build Command: leer. Details: SETUP.md §9.
